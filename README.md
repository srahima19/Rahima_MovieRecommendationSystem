# 🎬 Rahima_MovieRecommendationSystem

An AI-Based Movie Recommendation System built with Python as part of the Monthly Assessment Project for the **Python for AI & Data Science** course.

---

## 📌 Project Overview

This system recommends personalized movies to users based on:
- **Age group** — age-restricted movies are automatically filtered out
- **Preferred genre** — only matching genres are shown
- **Movie ratings** — only highly rated movies (7.5+) are recommended
- **User mood** — mood-matched movies are prioritized at the top

---

## 🚀 How to Run

1. Make sure Python 3 is installed on your machine
2. Clone or download this repository
3. Run the script in your terminal:

```bash
python Rahima_MovieRecommendationSystem.py
```

4. Follow the prompts — enter your name, age, preferred genre, and optional mood

---

## 🧠 Topics Used

| Topic | Usage |
|---|---|
| Variables | Storing user input and movie data |
| Conditional Statements | Age filtering, genre matching, rating threshold |
| For Loops | Traversing dataset, sorting, displaying results |
| Lists | Storing the movie dataset and recommendations |
| Tuples | Storing individual movie records |
